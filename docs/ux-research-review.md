## ภารกิจที่แนะนำ (30–45 นาที)

### T1 — ค้นหาดีไซเนอร์จากฟีด
1. เปิดหน้าแรก `/`
2. สลับแท็บ ผลงาน / ดีไซเนอร์ / สตูดิโอ
3. เปิดโปรไฟล์และผลงาน 2–3 รายการ

**สำเร็จเมื่อ:** เข้าใจว่าฟีดช่วยค้นหาและประเมินครีเอเตอร์ได้เร็วแค่ไหน

### T2 — ล็อกอินและจัดการพอร์ตโฟลิโอ
1. Login ด้วย `phatsawut@demo.an1hem.app`
2. ไป `/portfolio` ดู Welcome checklist
3. (ถ้าต้องการ) แก้ bio หรือลอง flow เพิ่มผลงาน

**สำเร็จเมื่อ:** รู้ว่าหลังล็อกอินทำอะไรต่อได้ทันที

### T3 — จ้างงาน / ขอคอลแลป
1. เปิดผลงานใดผลงานหนึ่งจากฟีด
2. กด **จ้างงาน** หรือ **ขอคอลแลป**
3. สังเกตฟอร์มและข้อความยืนยัน

**สำเร็จเมื่อ:** ไม่สับสนระหว่างจ้าง vs คอลแลป

### T4 — สำรวจงานและชุมชน
1. ไป `/jobs`
2. อ่านประกาศจ้าง / โหมดหางาน
3. กลับฟีด ลองไลก์หรือคอมเมนต์ (ถ้าล็อกอินแล้ว)

**สำเร็จเมื่อ:** เห็นภาพรวมชุมชนครีเอทีฟไทย

## สิ่งที่อยากได้ Feedback

- ภาษาไทยอ่านง่ายไหม — คำศัพท์ tech / ฟรีแลนซ์
- คอนเซปต์ **"ทุกคนคือ 1 PX"** สื่อสารได้หรือยัง
- ลำดับขั้นตอน จ้างงาน / คอลแลป / โพสต์งาน
- ความต่าง **1PX** (หน้าร้านโชว์งาน) vs **So1o** (หลังบ้านจัดการงาน)
- Mobile (375px) vs Desktop (1280px+) — จุดที่ใช้ยาก
- Empty state / loading / error — เข้าใจไหม
